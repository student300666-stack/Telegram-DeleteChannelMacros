# Telegram-DeleteChannelMacros
Этот простенький скрипт (макрос) поможет вам в один клик избавиться от лишних ТГК или чатов.
This simple script (macro) will help you get rid of unnecessary telegram channels or chats in one click.
# Установка
В терминале Windows (PowerShell) пропишите команду:
pip install pyautogui keyboard

In the Windows (PowerShell) terminal, enter the following command:
pip install pyautogui keyboard

# Использование
Данный макрос работает только в Telegram Desktop. Выберите лишний ТГК (либо чат) и открыв его жмите DELETE. А далее макрос сделает все за вас!
This macro only works in Telegram Desktop. Select an extra TG (or chat) and open it, then click DELETE. The macro will do the rest!
